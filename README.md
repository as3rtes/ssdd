# Learn the guitar

**Обучающее приложение для игры на гитаре, предназначенное для начинающих.**

Данное приложение позволяет изучать песни, просматривая тексты, аккорды и схему боя

## Возможности
- Выбор треков из базы данных с фильтрацией по исполнителю, названию и длительности
- Отображение текстов с аккордами
- Регулировка громкости
- Воспроизведение треков 
- Пауза и перемотка на 5 секунд
- Анимация нажатия кнопок

## Установка
### Необходимые требования:
- Python 3.12
- Git

### Шаги
1. Клонируем репозиторий Git
```bash
git clone https://github.com/96Leon27/Learn_the_guitar.git
cd Learn_the_guitar
```

2. Создаем и активируем виртуальное окружение
```bash
python -m venv venv

venv\Scripts\activate.bat или venv\bin\activate.bat (Для Windows)
source/bin/activate (Для Linux/macOS)
```

3. Установка необходимых библиотек
```bash
pip install -r requirements.txt
```

4. Запуск приложения
```bash
python main.py
```

## Как пользоваться
1. На главном экране выбирите "Старт"
2. В таблице выбирите трек по его ID
3. Введите этот Id в поле "song id" и нажмите "Начать"
4. В окне практики:
    - Нажмите ▶ для начала воспроизведения
    - Используйте ⟳ / ⟲ для перемотки
    - Показывайте/скрывайте аккорды и бой соответствующими кнопками
    - Регулируйте громкость ползунком
5. Для выхода из практики нажмите "Выйти"

## Структура проекта
```txt
Learn_the_guitar/
├── accords/          # текстовые файлы с аккордами
├── battle/           # схемы боя
├── songs_audios/     # MP3-файлы треков
├── songs_settings/   # настройки (связь с аккордами и боем)
├── songs_texts/      # тексты песен
├── ui_file_*.py      # сгенерированные UI-файлы (PyQt6)
├── main.py           # основной код приложения
├── tracks.sqlite     # база данных треков
├── requirements.txt  # зависимости
└── README.md         # документация
```
